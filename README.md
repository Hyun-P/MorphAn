# MorphAn

### Requirements
  * Python 3.8
  * Anaconda
  * FIJI (FIJI Is Just ImageJ)
  * Vaa3D
  * GCut
    * https://www.nature.com/articles/s41467-019-09515-0

## Installation
  * git clone https://github.com/Hyun-P/MorphAn.git
  * cd MorphAn
  * git clone https://bitbucket.org/muyezhu/gcut.git
  * conda env create -f environment.yml
